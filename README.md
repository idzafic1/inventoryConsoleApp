# \## Opis projekta

# Ovaj projekat predstavlja jednostavnu konzolnu aplikaciju za upravljanje inventarom. Fokus implementacije bio je na razvoju osnovnih funkcionalnosti sistema za evidenciju i manipulaciju podacima o inventaru.

# 

# Pored same aplikacije, poseban naglasak stavljen je na primjenu različitih \*\*white-box testnih tehnika\*\* s ciljem analize i poboljšanja kvaliteta koda. Testiranje je obuhvatilo provjeru unutrašnje logike sistema, uključujući pokrivenost grana, uslova i putanja izvršavanja.

# 

# \## Cilj projekta

# \- Razviti osnovnu konzolnu aplikaciju za upravljanje inventarom  

# \- Primijeniti i demonstrirati različite white-box testing tehnike  

# \- Analizirati kvalitet implementacije kroz testove
